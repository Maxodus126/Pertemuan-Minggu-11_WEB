📖 Konsep RAG (Retrieval-Augmented Generation) Sederhana:
1. Chunking: Dokumen dipecah menjadi potongan kecil dengan metadata (sumber, heading, indeks).
2. Retrieval: Query dicocokkan dengan chunk menggunakan keyword scoring — tanpa vector/embedding.
3. Grounding: Jawaban hanya berasal dari teks dokumen, tidak mengarang di luar sumber.
4. Sumber: Setiap jawaban menyertakan referensi nama file dokumen asal.
